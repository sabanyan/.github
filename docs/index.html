StreamCatとは？

StreamCatは、ノーコードでデータ前処理を構築できるプラットフォームです。製造装置のログデータやExcelデータを整え、機械学習の入力として活用可能にします。

GitHubリポジトリ

主な機能

  • フローエディタ：データ処理を視覚的に構築
  • ライブラリ：データの保管と管理
  • グラフ：データを視覚化
  • プロジェクト管理：コラボレーション機能
  • ユーザ管理：権限付与と管理

使い方

1. ユーザ登録を行います。

2. データをライブラリにアップロードします。

3. フローエディタで前処理を作成します。

4. グラフ機能でデータを可視化します。

5. プロジェクトを作成し、チームメンバーと協力して作業します。
